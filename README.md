# 👄 LipSync Assistant

A lip articulation prototype for ESL learners, lipreading learners, 
and people with auditory processing challenges.

## What it does
LipSync Assistant takes any word or sentence you type and displays 
animated face illustrations showing the correct mouth position for 
each sound. Each face is labeled with a plain-English description 
of the mouth shape, helping users connect written letters to their 
physical pronunciation.

## Why I built it
Inspired by conversations with someone close to me who has Auditory 
Processing Disorder, and validated through user research with 
lipreading and ESL communities. The goal is to make pronunciation 
more accessible and visual — especially for faceless video content 
where there's no mouth to read.

## How to run it
1. Clone this repo
2. Install dependencies: `pip install streamlit`
3. Run the app: `streamlit run app.py`
4. Open your browser to `http://localhost:8501`
